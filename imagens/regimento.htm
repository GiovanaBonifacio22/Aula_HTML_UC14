Regimento Escolar

* Nota média para aprovação: 7,0. Essa média é alcançada através aritmética das 4 notas bimestrais.
* Frequencia para aprovação: 75%. Faltas justificadas através de requisição apresentada pelo aluno serão abonadas.

O Conselho escolar é soberano para definir a aprovação mesmo que os resultados não tenham sido alcançados.
